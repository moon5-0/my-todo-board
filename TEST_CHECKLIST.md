## 一、功能测试

### 1.1 添加任务
- [❎] 在顶部「添加任务」输入标题，选择列表，点击添加 → 任务出现在对应列表
- [❎] 在某个列表内点击「+ 添加任务」→ 输入框出现在该列表底部
- [❎] 在列表内输入内容按 Enter → 任务添加成功
- [❎] 空内容提交 → 不应添加任务

### 1.2 删除任务
- [❎] 点击任务上的删除图标 → 弹出确认弹窗
- [❎] 点击弹窗「取消」→ 弹窗关闭，任务仍在
- [❎] 点击弹窗「删除」→ 任务被移除
- [❎] 点击弹窗外部遮罩 → 弹窗关闭

### 1.3 编辑任务
- [❎] 点击任务的编辑图标 → 标题变为输入框
- [❎] 修改内容后按 Enter → 保存成功
- [❎] 按 Escape → 取消编辑，恢复原内容
- [❎] 点击空白处 → 自动保存并退出编辑

### 1.4 标记完成/未完成
- [❎] 点击任务复选框 → 任务显示删除线 + 灰色文字
- [❎] 再次点击复选框 → 恢复未完成状态

### 1.5 数据持久化
- [❎] 添加几个任务后，刷新页面（F5）→ 任务仍在
- [❎] 关闭浏览器，重新打开页面 → 数据仍在
- [❎] 删除任务后刷新 → 删除的任務不出现

### 1.6 深色模式
- [❎] 点击右上角月亮/太阳图标 → 主题切换（背景/文字颜色变化）
- [❎] 刷新页面 → 主题保持不变
